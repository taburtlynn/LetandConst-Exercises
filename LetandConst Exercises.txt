- What is the difference between ***var*** and ***let***?
can redeclare var but not let. var is function scope and let is block scope.

- What is the difference between **var** and ***const***?
const cannot be redeclared or reassigned. var is function scope and const is block scope.

- What is the difference between **let** and **const**?
let can be reassigned.

- What is hoisting?
var variable declaration is run before initialization, comes back undefined but no error.
